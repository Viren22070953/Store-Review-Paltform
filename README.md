# Store Rating Platform

A full-stack web application where users can rate stores, store owners can view ratings, and administrators can manage users and stores.

## Features

- User registration and login
- Role-based dashboard for User, Store Owner, and Admin
- Users can browse stores and submit ratings
- Store owners can view customer feedback
- Admin can manage users and stores
- Responsive and modern UI

## Tech Stack

- React.js
- React Router
- Node.js
- Express.js
- MySQL 
- CSS
